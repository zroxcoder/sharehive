üöÄ Getting Started with ShareHive

Welcome to ShareHive! Here's everything you need to know to begin your productivity journey.

üí° What is ShareHive?

ShareHive is a personal productivity hub that stores all your data locally in your browser. No account needed, completely private, and works offline!

Quick Start Steps:

  1. Explore the Sidebar: Navigate through different features like Notes, Bookmarks, To-Do List, and more from the left sidebar.
  2. Create Your First Item: Click on any feature and start creating notes, tasks, or bookmarks immediately.
  3. Customize Your Experience: Toggle dark mode for comfortable viewing and adjust settings to your preference.
  4. Your Data is Safe: Everything is stored locally in your browser using IndexedDB technology.
  5. Backup Regularly: Export your data from Settings to keep a backup file safe.

‚ÑπÔ∏è Important to Know

ShareHive runs entirely in your browser. Clearing browser data or using incognito mode will result in data loss. Always export your data as backup!

üíæ Data Management & Backup

Understanding how ShareHive stores and manages your data:

How Your Data is Stored:

‚ÑπÔ∏è Local Storage

ShareHive uses IndexedDB, a browser-based database system. All your data stays on your device and never leaves your browser. This means:

  • ‚úÖ Complete privacy - no server storage
  • ‚úÖ Works offline without internet
  • ‚úÖ Fast access to your data
  • ‚ö†Ô∏è Data is tied to your browser and device
  • ‚ö†Ô∏è Clearing browser data will delete everything

Backing Up Your Data:

  1. Open Settings: Click on "‚öôÔ∏è Settings" in the sidebar
  2. Find Export Option: Look for the "Export Data" or "Backup" section
  3. Download Backup: Click "Export" to download a JSON file with all your data
  4. Save Safely: Store the backup file in a safe location (cloud storage, external drive, etc.)
  5. Regular Backups: Repeat this process weekly or after important updates

Restoring Your Data:

  1. Open Settings: Navigate to Settings page
  2. Find Import Option: Look for "Import Data" or "Restore" section
  3. Select File: Click "Import" and choose your backup JSON file
  4. Confirm Import: Review the import details and confirm
  5. Refresh Page: Reload the page to see your restored data

‚ö†Ô∏è Important Warnings

  • Importing data may overwrite existing data - backup first!
  • Using incognito/private mode prevents data persistence
  • Browser storage has limits (typically 50-100MB)
  • Uninstalling your browser will delete all data
  • Switching devices requires manual export/import

üîß Troubleshooting & Common Issues

Solutions to frequently encountered problems:

Possible Causes:

  • Browser cache or data was cleared
  • You're using incognito/private browsing mode
  • Browser storage quota was exceeded
  • Browser was uninstalled/reinstalled
  • You switched to a different browser or device

Solution:

Unfortunately, if data is cleared from IndexedDB, it cannot be recovered unless you have a backup. This is why regular exports are crucial!

‚ö†Ô∏è Prevention

Always maintain backup files by exporting your data weekly. Store backups in multiple locations (cloud storage, email, external drive).

Common Causes:

  • Too much data stored (thousands of items)
  • Large files uploaded to File Manager
  • Too many browser tabs or extensions running
  • Outdated browser version
  • Low device memory/storage

Solutions:

  1. Archive or delete old, unused data
  2. Remove large uploaded files
  3. Close unnecessary browser tabs
  4. Update to the latest browser version
  5. Clear browser cache (but don't clear site data!)
  6. Try using a different modern browser (Chrome, Firefox, Edge)

ShareHive currently doesn't support automatic cloud synchronization. However, you can manually sync using this method:

Manual Sync Process:

  1. On Device 1: Export your data from Settings ‚Üí Download backup file
  2. Transfer: Send the backup file to Device 2 (email, cloud storage, USB, etc.)
  3. On Device 2: Import the backup file via Settings ‚Üí Import Data
  4. Regular Updates: Repeat this process whenever you need to sync

‚ÑπÔ∏è Future Feature

Cloud sync and multi-device support are planned for future updates. This would allow automatic synchronization across all your devices!
